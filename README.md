# Bicycle routes tester

## Introduction

Bicycle routes tester is an application for android, that works with arduino..

## Description

Arduino uses the accelerator MPU6050 and Bluetooth HC 05 connected to the serial port, the project includes the necessary libraries.
An application written in Android Studio in Java language. Ap uses Google Maps Api, GPS, and a sqlite database

## Installation

You should upload the arduino program, then run the applications in the android studio and rebuild the gradle
